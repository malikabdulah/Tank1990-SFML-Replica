# Tank 1990 Replica (C++ & SFML)

A 2D top-down arcade shooter inspired by the classic retro game *Tank 1990*. This project was developed as my first-semester Programming Fundamentals (PF) project at FAST-NUCES. 

While it captures the core mechanics and nostalgic feel of the original game, it is a custom adaptation rather than a strict 1:1 clone. 

## 🎮 Gameplay Features
* **Classic Tank Movement:** Omnidirectional grid-based movement.
* **Combat Mechanics:** Projectile shooting and collision detection.
* **Custom Game Logic:** Implemented entirely in C++ utilizing foundational programming concepts.
* **2D Graphics:** Visuals and rendering powered by the Simple and Fast Multimedia Library.

## 🛠️ Tech Stack
* **Language:** C++
* **Graphics Library:** SFML 2.5.1 (Simple and Fast Multimedia Library)
* **Concepts Utilized:** Game loop architecture, event handling, memory management, and collision logic.
